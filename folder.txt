Key APIs to Build

Auth APIs: Register, Login, Refresh Token, Forgot Password
User APIs: Get Profile, Update Profile, Upload Profile Photo
Photo APIs: Upload, Fetch by Place, Fetch User Photos, Delete
Google Photos APIs: Authenticate, Fetch Albums, Sync Album
Watermark APIs: Get Settings, Update Settings (admin only)
Places APIs: Get All Places, Get Place Details, Search Places
Blog APIs: Create, Read, Update, Delete, List by Place
Wallet APIs: Get Balance, Get Transactions, Redeem
Admin APIs: Approve/Reject Photos, Manage Users, Analytics
